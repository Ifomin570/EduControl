🚀 КАК СОЗДАТЬ ИГРУ:

1. Попроси нейросеть написать игру по шаблону

2. Нажми кнопку ПРОМТ ниже чтобы скопировать шаблон

3. Вставь идею игры в [ВСТАВЬТЕ СВОЮ ИДЕЮ ЗДЕСЬ]

4. Добавь иконку (ссылка или смайлик)

5. Нажми INSTALL и вставь код

⚡ ФОРМАТ ИГРЫ:

{
  name: "НАЗВАНИЕ",
  icon: "URL/🎮",
  init: function() { ... },
  update: function() { ... },
  draw: function(ctx) { ... }
}

🎯 ОСОБЕННОСТИ:

• backToMenu() - выход в меню
• window.onkeydown для клавиш
• 640x480 canvas
• ctx.clearRect(0,0,640,480)
